# 🌤️ SkyCheck — Weather App

A real-time weather web app that shows live conditions for any city in the world, with dynamic animated backgrounds and weather-based advice notifications.

Built as part of my ICT coursework using vanilla **HTML, CSS, and JavaScript**.

## ✨ Features

- 🔎 Search weather for any city worldwide
- 🌡️ Real-time temperature, condition, and wind speed
- 🎨 Animated background that changes based on the weather (sun, clouds, rain, lightning)
- 🌈 Animated rainbow border around the weather widget
- 💡 Smart popup notifications with weather-based advice (e.g. "Stay hydrated" on hot days, "Keep your umbrella with you" on rainy days)
- 📱 Responsive design

## 🛠️ Built With

- HTML5
- CSS3 (animations, gradients, responsive design)
- JavaScript (DOM manipulation, async/await, fetch API)
- [Open-Meteo API](https://open-meteo.com/) — free weather and geocoding data (no API key required)
